== Description ==

This plugin allows to add a Ringotel webchat widget to your website and talk to your website visitors in real-time.

You can find more information on [Ringotel website](https://ringotel.co)

If you find any bugs/issues please report and I'll try to fix them asap.

**How to make this plugin work?**

While Ringotel's webchat plugin for WordPress is free, you need a subscription of Ringotel to use it. For an overview of available plans and their cost navigate to Ringotel [pricing](https://ringotel.co/pricing).

[Sign up for Ringotel here](https://ringotel.co/signup)

Ringotel's [Terms of Service](https://ringotel.co/terms-of-service/) & [Privacy Policy](https://www.iubenda.com/privacy-policy/92894652)


**How does this plugin works?**

* This plugin adds a Ringotel webchat widget to your website.


== Installation ==


**via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'Ringotel Webchat Plugin'
2. Click 'install'

**Manual Installation**

1. Unzip the zip file and upload  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu
3. Configure plugin from 'Settings > PS Menu Items'
